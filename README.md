# ScoreBoardApp

Sports Day **live scoreboard** built with React and Vite: team totals, per-sport slides, admin score entry, and fullscreen display.

## Quick start

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
| -------- | ------------- |
| `npm run dev` | Start dev server |
| `npm run build` | Production build to `dist/` |
| `npm run preview` | Preview production build |

## Tech

- React 19 · Vite 8 · react-router-dom

Scores are stored in `localStorage` in the browser unless changed.
